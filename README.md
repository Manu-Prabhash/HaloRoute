# HaloRoute
Halo Route Smart Tourist Safety Monitoring & Incident Response System
Overview
Halo Route is a real-time safety monitoring platform designed to protect travelers in unfamiliar areas. By integrating live location tracking with geo-fencing and automated alerts, the system provides a digital "Guardian" for users, ensuring rapid response during emergencies.

Key Features
Live Map Integration: Real-time tracking with interactive markers for location monitoring.

Guardian Module: Core safety logic designed to trigger alerts when users deviate from safe routes.

Geo-Fencing: Dynamic "Red Zone" detection that notifies users and emergency contacts upon entry.

Instant Incident Response: A dedicated Panic Button for immediate emergency signaling.

Responsive Dashboard: Sleek UI with Dark/Light mode support for high visibility in any condition.

Tech Stack
Frontend: HTML5, CSS3, JavaScript (ES6+)

Mapping: Leaflet.js / Google Maps API

Backend: Node.js / Python (Flask/Django)

Database: Firebase / MongoDB
